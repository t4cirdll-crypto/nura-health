# Nura Health Landing Page

A cinematic React 19 + Tailwind CSS + GSAP landing page for a high-end organic tech / clinical boutique brand.

## Run

```bash
npm install
npm run dev
```

## Stack

- React 19
- Vite
- Tailwind CSS via `@tailwindcss/vite`
- GSAP 3 + ScrollTrigger
- Lucide React

## Design

Palette: Moss `#2E4036`, Clay `#CC5833`, Cream `#F2F0E9`, Charcoal `#1A1A1A`.
